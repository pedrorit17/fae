🔥 Firebase (Jogo Multiplayer)

Configure o Firebase Realtime Database para o modo multiplayer do Jogo da Vida Empresarial. Essa seção é opcional — o jogo funciona em modo local sem Firebase.

  1. Acesse o Firebase Console e crie um projeto (ou use um existente).
  2. Vá em Realtime Database → Criar banco e escolha o modo teste.
  3. Vá em Configurações do projeto → Geral e copie os dados do app web abaixo.

Credenciais do Firebase

Copie os valores de firebaseConfig do console.

Publicar configuração

  1. Teste a conexão das duas planilhas acima.
  2. Clique em Baixar config.json.
  3. Substitua data/config.json no repositório e faça commit + push.
  4. No dia a dia: edite as planilhas diretamente — as páginas atualizam automaticamente.
